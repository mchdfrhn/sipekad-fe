# 🚀 SIPEKAD - Sistem Pengajuan Akademik

[![React](https://img.shields.io/badge/React-19-blue.svg)](https://reactjs.org/)
[![Vite](https://img.shields.io/badge/Vite-7-purple.svg)](https://vitejs.dev/)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind-4-blue.svg)](https://tailwindcss.com/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**SIPEKAD** adalah platform dashboard administrasi modern yang dirancang untuk mengelola berbagai pengajuan akademik secara efisien. Dibangun dengan teknologi web terkini untuk memberikan pengalaman pengguna yang mulus, responsif, dan premium.

---

## ✨ Fitur Utama

-   **📊 Dashboard Analytics**: Visualisasi data real-time menggunakan ApexCharts dan Recharts untuk memantau statistik pengajuan.
-   **📝 Manajemen Pengajuan**: Sistem pelacakan dan pemrosesan pengajuan akademik yang terorganisir.
-   **👥 Manajemen Pengguna**: Kontrol akses dan profil pengguna yang komprehensif.
-   **🤖 Integrasi WhatsApp**: Bot otomatis untuk notifikasi dan manajemen komunikasi melalui WhatsApp.
-   **💾 Backup & Security**: Sistem backup data otomatis dan keamanan tingkat tinggi.
-   **🌗 UI Premium**: Desain modern dengan sentuhan *Glassmorphism*, animasi halus (*Framer Motion*), dan komponen *Radix UI*.

---

## 🛠️ Tech Stack

### Frontend Core
-   **React 19**: Library UI paling populer dengan performa maksimal.
-   **Vite 7**: Build tool super cepat untuk pengembangan modern.
-   **Tailwind CSS 4**: Engine styling utility-first terbaru untuk desain yang presisi.

### UI & UX
-   **Radix UI**: Komponen primitif aksesibel untuk fondasi UI yang kokoh.
-   **Lucide React**: Set ikon vektor yang bersih dan konsisten.
-   **Motion (Framer Motion)**: Library animasi tingkat lanjut untuk interaksi yang hidup.

### Data & Logic
-   **React Router 7**: Routing aplikasi yang handal dan fleksibel.
-   **Axios**: Penanganan request API yang robust.
-   **ApexCharts & Recharts**: Visualisasi data yang interaktif dan informatif.

## 🚀 Memulai (Getting Started)

### Prasyarat
-   Node.js (versi terbaru direkomendasikan)
-   NPM atau Yarn

### Instalasi

1.  **Clone repositori ini:**
    ```bash
    git clone https://github.com/username/sipekad-fe.git
    cd sipekad-fe
    ```

2.  **Instal dependensi:**
    ```bash
    npm install
    ```

3.  **Konfigurasi Environment:**
    Salin file `.env example` menjadi `.env` dan sesuaikan nilainya.
    ```bash
    cp ".env example" .env
    ```

4.  **Jalankan aplikasi (Mode Pengembangan):**
    ```bash
    npm run dev
    ```
